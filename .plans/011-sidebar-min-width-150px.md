# サイドバー最小幅を 150px に変更

## 概要

サイドバーのリサイズ最小幅を `200px` から `150px` に変更する。
あわせて初期幅も `20%` から `150px` に変更する。
再起動後の幅保存は対象外とする。

## 変更内容

- `ui/components/Preview.tsx` の `Resizable` に指定されている `minWidth` を `150px` に変更する。
- `ui/components/Preview.tsx` のサイドバー初期幅を `150px` に変更する。
- `ui/style.css` の `nav` に指定されている `min-width` を `150px` に変更する。
- `ui/components/SideBar.tsx` の見出しテキストを折り返さず、横幅を超える場合は省略表示にする。
- 必要に応じてバンドル済みアセットを再生成し、実行時の表示も同じ値に揃える。

## 確認内容

- `npm run lint` を実行し、静的チェックが通ることを確認する。
- バンドル再生成後、生成済み CSS/JS にも `150px` が反映されていることを確認する。
