# サイドバー見出しのパディングをさらに減らす

## 1. 課題

サイドバーの見出しパディングを一段詰めたが、全体的にまだ余白が広く見える。
左右の余白と階層ごとのインデント量をもう少し減らしたい。

## 2. 対象箇所の確認

- `ui/components/SideBar.tsx` の `buttonStyle.padding` で見出し項目の余白を指定している。
- 現状は `padding: 0 1em 0 ${h.level - 1}em` で、右余白が `1em`、レベル2の左余白が `1em` になっている。

## 3. 仕様整理

- レベル1見出しは引き続き非表示にする。
- レベル2見出しの左余白をさらに減らす。
- レベル3以降も階層差は残すが、1階層あたりの増加量を小さくする。
- 右余白も少し減らす。

## 4. 実装方針

- 左余白を `0.5 + (h.level - 2) * 0.75` em にする。
    - レベル2: `0.5em`
    - レベル3: `1.25em`
    - レベル4: `2em`
- 右余白を `0.75em` にする。
- 計算値を `indent` 変数に分け、padding の意図を読みやすくする。

具体的なコード案:

```ts
const indent = 0.5 + (h.level - 2) * 0.75;
const buttonStyle = {
    padding: `0 0.75em 0 ${indent}em`,
};
```

## 5. 影響確認

- サイドバー見出しの余白が全体的に詰まることを確認する。
- 階層ごとのインデント差が残ることを確認する。

## 6. 動作確認方針

- TypeScript の型チェックを実行する。
- Prettier と ESLint を実行する。

## 7. 対応内容 (結果)

- `SideBar` の見出し左余白を `0.5 + (h.level - 2) * 0.75` em に変更した。
- 右余白を `1em` から `0.5em` に減らした。
- レベル2以降の階層差を残しつつ、サイドバーの見出し余白を全体的に詰めた。
