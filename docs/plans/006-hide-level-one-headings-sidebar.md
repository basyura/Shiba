# サイドバーでレベル1見出しを非表示にする

## 1. 課題

サイドバーの見出し一覧に Markdown の `#` 見出し、つまりレベル1見出しが表示されている。
サイドバーにはレベル2以降の見出しのみを表示したい。

## 2. 対象箇所の確認

- `ui/components/SideBar.tsx` の `SideBar` コンポーネントで `headings.map` により見出し一覧を生成している。
- 現状は `Heading.level` に関係なくすべての見出しを表示している。

## 3. 仕様整理

- `Heading.level === 1` の見出しはサイドバーに表示しない。
- レベル2以降の見出しは従来通り表示し、クリック時のスクロール挙動も維持する。
- サイドバー上部のファイル名ヘッダーは対象外とし、従来通り表示する。

## 4. 実装方針

- `headings` を map する前に `level > 1` で絞り込む。
- 既存の `ListItem` 描画処理は維持し、表示対象だけを変更する。

具体的なコード案:

```tsx
const children = headings.filter((h) => h.level > 1).map((h, key) => {
    ...
});
```

## 5. 影響確認

- レベル1見出しがサイドバーに表示されないことを確認する。
- レベル2以降の見出しの表示、選択状態、クリックスクロールが従来通り動作することを確認する。

## 6. 動作確認方針

- TypeScript の型チェックを実行する。
- ESLint を実行する。

## 7. 対応内容 (結果)

- `SideBar` の見出し一覧生成時に `level > 1` の見出しだけを表示するようにした。
- サイドバー上部のファイル名ヘッダーや、レベル2以降の見出しの描画処理は維持した。
